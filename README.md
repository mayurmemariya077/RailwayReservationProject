## 📖 Project Overview

The **Railway Reservation System** is a console-based C++ application developed using **Object-Oriented Programming (OOP)** concepts. This system allows users to manage train records efficiently by adding new train details, displaying all available train records, and searching for trains using their train number.

The project demonstrates the practical implementation of **Classes, Objects, Constructors, Destructors, Static Data Members, Encapsulation, and Arrays of Objects** in C++.

---

## ✨ Features

✅ Add New Train Records
✅ Display All Train Records
✅ Search Train by Train Number
✅ Menu Driven Interface

✅ Easy to Understand and Maintain

| ---------------- | ----------------------------------- |
| Class            | `Train`, `RailwaySystem`            |
| Object           | Train Objects                       |
| Constructor      | Default & Parameterized Constructor |
| Destructor       | Object Cleanup                      |
| Encapsulation    | Private Data Members                |
| Static Member    | `trainCount`                        |
| Array of Objects | `Train trains[100]`                 |

## 📂 Class Structure

### 🚂 Train Class

Stores train-related information:

* Train Number
* Train Name
* Source
* Destination
* Train Time

Functions:

* `inputTrainDetails()`
* `displayTrainDetails()`
* `getTrainNumber()`
* `getTrainCount()`

---

### 🚉 RailwaySystem Class

Manages all train records.

Functions:

* `addTrain()`
* `displayAllTrains()`
* `searchTrainByNumber()`
